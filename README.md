## Blog Workflow

1. Write and review drafts in [aintropy-blog](https://github.com/aintropy-ai/aintropy-blog)
2. Share via `file:///` path in browser for internal review
3. Get approval from Kunal
4. Copy approved post into Demo_walkthrough and push to go live on aintropy.ai
